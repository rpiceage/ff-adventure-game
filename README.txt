Fighting Fantasy Adventure Game
================================

How to run:
-----------
java -jar ff-1.0-jar-with-dependencies.jar

This will start the complete demo adventure with all features.

To load a custom adventure:
java -jar ff-1.0-jar-with-dependencies.jar path/to/your-adventure.yaml

Features:
---------
- Turn-based combat (single and multiple enemies)
- Luck tests
- Item collection and inventory
- Gold management
- Attribute modifications (SKILL, STAMINA, LUCK, GOLD)
- Animated dice rolls
- Internationalization (English/Hungarian)

Requirements:
-------------
Java 11 or higher
